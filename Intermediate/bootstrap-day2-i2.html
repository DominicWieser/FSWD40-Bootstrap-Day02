html
HTML is the HyperText Markup Language for describing web pages
css
CSS stands for Cascading Style Sheet. CSS is based on a set of rules that describe the priority of how styles are rendered on the page. The rules can be as simple as defining colors, sizes, fonts, alignment etc. The newest version of CSS - CSS3 goes beyond the basic rules offering animation and even a level of interactivity.
javascript
JavaScript is a programming language used to make web pages interactive. In conjunction with HTML 5 and CSS 3 you will be able to create rich, interactive and dynamic GUI (Graphic User Interface)
json
JSON or JavaScript Object Notation is a lightweight text-based open standard designed for human-readable data interchange. The JSON filename extension is .json.
jqurey
JQuery is a library that makes writing JavaScript code a lot easier, with a nice motto: “Write less, do more”. It helps you build cross-browser websites or applications.
bootstrap
In the earlier days of Twitter, engineers used almost any library they were familiar with to meet front-end requirements. Inconsistencies among the individual applications made it difficult to scale and maintain them. Bootstrap began as an answer to these challenges and quickly accelerated during Twitter’s first Hackweek. By the end of Hackweek, we had reached a stable version that engineers could use across the company. — Mark Otto
